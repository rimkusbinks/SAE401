API REST SAE 4.01

Endpoints pour interroger la base de données

Récupération des Stations

GET /stations

Récupération des Polluants

GET /pollutants

Récupération des Centres Régionaux

GET /regional_centers

Récupération des Mesures pour une Station Spécifique

GET /measurements?station_id={station_id}

Récupération des Mesures pour un Polluant Spécifique

GET /measurements?pollutant_id={pollutant_id}

Récupération des Mesures pour une Période Spécifique

GET /measurements?station_id={station_id}&start_date={yyyy-mm-dd}&end_date={yyyy-mm-dd}

Récupération de la Moyenne Journalière pour un Polluant Spécifique

GET /daily_average?pollutant_id={pollutant_id}&date={yyyy-mm-dd}

Récupération de la Moyenne Journalière pour tous les Polluants sur une Station

GET /daily_average?station_id={station_id}&date={yyyy-mm-dd}

Récupération du Maximum et Minimum pour un Polluant Spécifique sur une Période

GET /range?pollutant_id={pollutant_id}&start_date={yyyy-mm-dd}&end_date={yyyy-mm-dd}

Récupération des Mesures pour une Station, un Polluant et une Date Précise

GET /measurements?station_id={station_id}&pollutant_id={pollutant_id}&date={yyyy-mm-dd}
